# LinkedIn Post — ShieldOps Launch

---

🛡️ I built a full SaaS cybersecurity platform from scratch — and it's live.

ShieldOps is a multi-tenant website security monitoring platform for small businesses. Think of it as a SOC (Security Operations Centre) as a service — but built for the Irish SMB market, not enterprise.

Here's what it does:

🔍 Monitors websites 24/7 for brute force attacks, page tampering, SSL expiry, uptime issues, and DNS changes

🤖 Uses AI (Google Gemini) to explain security alerts in plain English — so a non-technical business owner can understand exactly what happened to their site

⚡ Gives analysts one-click response actions — block IPs via Cloudflare, force WordPress password resets, rate-limit endpoints — with a customer approval workflow built in

📋 Generates compliance documents on demand — GDPR breach notifications, ISO 27001 evidence packs, and cyber insurance claim reports

The platform has 3 separate portals:
→ Customer Portal (for the business owner)
→ SOC Analyst Console (for the security team)
→ Admin Console (for platform management)

Tech stack: React 18 + Vite, Firebase (Auth + Firestore), Google Gemini API, Recharts, jsPDF — hosted free on Netlify.

The whole thing — auth, multi-tenancy, AI chat, real-time approval workflow, shared threat intel feed, MITRE ATT&CK mapping — runs at €0/month in infrastructure costs until it scales.
